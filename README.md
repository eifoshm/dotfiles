dotfiles
========

This repo contains (almost) all my current configs.

* bspwm: WM config
* sxhkd: keybindings
* ncmpcpp: music player setup
* scripts: all my scripts (+ bspwm panel executables)
* xorg: visual stuff (colors + urxvtd/urxvtc)
